# USB 介绍

 USB（通用串行总线）是一个外部总线标准，规范电脑与外部设备的连接和通讯，其内部只有4根线，两根电源线两根信号线。USB支持热插拔（机器在运行中插入usb设备可以直接与该设备通信，无需将电脑重启）。

USB的发展历程从USB 1.0开始，到后面的USB1.1，USB 2.0，USB 3.0，USB 3.1，USB 3.2，直到USB 4.0，不断提升了数据传输速度和设备兼容性，广泛应用于各种电子设备中。
